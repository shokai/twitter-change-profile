Twitter Change Profile
======================
update profile using wikipedia


Install
=======

    % git clone git://github.com/shokai/twitter-change-profile.git
    % cd twitter-change-profile


Install dependencies
====================

    % gem install bundler
    % bundle install


Config
======

    % tw --user:add

edit config.yaml.


Update Your Profile
===================

    % bundle exec ruby change-profile.rb
